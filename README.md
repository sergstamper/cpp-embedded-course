# C++ Embedded / MilTech Engineering Course

A structured, project-based journey to become a professional C++ Embedded Engineer.

## Objectives

- Master Modern C++20
- Learn professional software engineering practices
- Build production-quality applications
- Explore Embedded Linux, STM32, ESP32, Qt, ROS2, and FreeRTOS

## Repository Structure

```text
.
├── lessons/
├── notes/
├── playground/
└── tools/
```

## Learning Approach

Each lesson contains:

- theory
- practical exercises
- code review
- best practices
- mini-projects

## Development Environment

- Ubuntu (WSL2)
- GCC
- Clang
- CMake
- Ninja
- Git
- Visual Studio Code
